## 🚀 快速开始

### 安装

```bash
# 安装到 Hermes Agent
hermes skills install https://github.com/woqianfu/duanju-master
```

### 使用

在 Hermes 对话中发送 `@短剧大师` 触发，然后按 Phase 1 → 2 → 3 → 4 流程操作。

```bash
# 搜索蒸馏案例库
python3 assets/short-writing/scripts/search_library.py "霸总 第一屏" --top 5

# 搜索完整剧本库（机制提取）
python3 assets/short-writing/scripts/search_full_scripts.py "杂灵根" --top 3

# 查看案例卡详情
python3 assets/short-writing/scripts/inspect_case.py c0001

# 提交到小云雀
export XYQ_ACCESS_KEY="your-access-key"
python3 scripts/submit_run.py --message "描述"
```

### 创作流程

1. **一句话创意** → Phase 1 产出 12 项交付物
2. **12 项交付物** → Phase 2 升级为 5/6 镜量化剧本
3. **量化剧本** → Phase 3 提交小云雀 API
4. **生成产物** → Phase 4 SQ 质量管控全闭环
