# Nano Banana 2 + 三视图 — 角色一致性实战方案

> 来源：Threads/CSDN/B站社区实战分享（2026.06）
> 吸收日期：2026-06-28

## 核心发现

**Nano Banana 2 + 角色三视图 → 可维持90%以上跨镜角色一致性。**

对比当前短剧大师的角色一致性方案（LoRA + Seed固定 + 外观关键词锁定），Nano Banana 2提供了另一种低成本路径：

| 方案 | 一致性 | 成本 | 适用场景 |
|------|:------:|:----:|---------|
| LoRA训练 | 95%+ | 几十元/角色（一次性） | 长期项目/多集连续剧 |
| Nano Banana 2 + 三视图 | 90%+ | 0元（prompt模式） | 快速试错/短项目 |
| 纯关键词锁定 | 60-70% | 0元 | 不推荐（已过时） |

## 三视图Prompt模板

```
一张【角色名】的全身三视图，白底，正面全身/侧面全身/背面全身。
【画风描述：3D国风/真人/动漫】
【外貌特征：脸型/五官/发型/发色】
【服装：逐件描述+颜色+材质】
【配饰：冠/钗/腰带/兵器等】
```

## 在短剧大师中的使用

1. 造型大师先按6维参数产出角色设计
2. 用此三视图prompt生成参考图
3. 将三视图作为Nano Banana 2的参考图注入视频生成
4. 每镜prompt中固定写入角色外观关键词 + `"based on reference image"`

## 限制

- Nano Banana 2对中文prompt的理解不如小云雀/即梦
- 三视图只能作为「参考垫图」，不能完全替代LoRA的精确控制
- 对高速运动/大角度转头的场景，一致性降到70-80%
