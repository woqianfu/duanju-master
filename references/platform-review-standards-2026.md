## 一、抖音AI短剧审片标准（2026.5发布）

### 1.1 四条绝对禁止

| # | 类别 | 禁止内容 |
|:--:|------|---------|
| ① | 不良价值观导向 | 宣扬拜金主义、极端复仇、性别物化、畸形婚恋、美化出轨或家暴 |
| ② | 低俗色情擦边 | 暴露画面、性暗示台词、低俗封面标题、以擦边内容引流变现 |
| ③ | 观感不适 | 血腥恐怖、恶心猎奇、无底线审丑、静态PPT式敷衍内容 |
| ④ | 侵权与肖像滥用 | 未经授权AI换脸、盗用动漫/游戏素材、模仿明星肖像、洗稿抄袭 |

### 1.2 覆盖面

审核覆盖**剧名、封面、简介、正片、配音、字幕、画质**，杜绝「标题擦边、内容违规」漏洞。

### 1.3 技术质量红线

- 禁止机械配音、音画不同步、人物穿模
- 禁止烂尾断更、水印外链
- 封面禁用「热播」「爆款」「强推」等营销字样
- 封面禁止视觉擦边或主体五官缺失

### 1.4 AI标识强制（三处标注）

| 位置 | 规范 |
|:----:|------|
| **片头前3秒** | 正片开始前，黑底白字/与正片融合，清晰可见≥2秒 |
| **封面** | 封面右上角或底部，字体不小于正文字号 |
| **简介** | 简介开头第一句 |

> AI生成内容占比≥**50%**的作品，须三处同时标注「本作品含AI生成内容」。未达标→**限流或下架**。

### 1.5 审核数据

| 指标 | 数值 |
|:----|:----:|
| 审片前日上新量 | 2000+部 |
| 审片后日上新量 | ≈350部 |
| 新剧过审率 | ≈30% |
| 精品激励 | 保底收益+高分成+核心流量倾斜+IP改编补贴+出海专项扶持 |
| 专项资金 | 2亿元（扶持现实主义、悬疑、都市群像等差异化内容） |

### 1.6 智能体工具特殊要求

| 要求 | 说明 |
|------|------|
| Seedance 2.0 | 限制真人图像/视频作为主体参考 |
| 真人使用 | 如需使用真人作为主体参考，须经本人验证或取得授权 |
| 工具合规 | 智能体工具须接入平台审核系统，实现「创作-审核-上线」全链路监控 |

## 二、红果短剧《动画微短剧（漫剧）内容创作建议》（2026.3发布·4.7执行）

### 2.1 核心红线

- 禁止危害国家安全、损害民族精神、违背公序良俗
- 禁止色情低俗、暴力血腥、恐怖惊悚内容
- 禁止「盗脸」「换脸」等侵犯肖像权行为
- 上述建议可通过抖音集团短剧版权中心查阅

### 2.2 平台规则变更

| 时间 | 变更 | 影响 |
|------|------|------|
| 2026.2 | 单账户日上传上限30部 | 限制批量低质内容 |
| 2026.3 | 红果发布漫剧内容创作建议 | 内容红线明确化 |
| 2026.4.1 | AI漫剧备案新规强制执行 | 持证上岗 |
| 2026.4月上旬 | 《菩提临世》下架（行业标志事件） | 合规警钟 |
| 2026.5 | 抖音发布AI短剧审片标准 | 系统性规范 |

### 2.3 精品漫剧激励政策（第二期）

- 高评级AI仿真人剧、2D/3D漫剧→对应保底激励
- 番茄小说原创头部IP→补贴100%漫剧改编权费用
- 阅文等头部IP方深度合作

## 三、创作者实操建议

### 3.1 剧本阶段
- 优先完成内容备案
- 素材合规审查（文字/图片/音乐授权）
- 建立素材授权档案，留存授权凭证

### 3.2 制作阶段
- 使用原创数字人形象+版权登记
- 优先选择平台认证的智能体工具
- 建立「AI生成→人工校对→二次修改」质量控制流程
- AI生成人物形象须与真人有显著区别

### 3.3 上线阶段
- 著作权登记（留存prompt修改过程/再创作/调整的确权留证）
- 植入数字水印（主流云服务平台都有提供）
- 舆情监测机制
