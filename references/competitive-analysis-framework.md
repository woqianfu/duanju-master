## 七、对照组（Comparative Group）设计规则

1. 对照组 ≠ 反派。对照组是「普通人的普通选择」，不是恶人。
2. 功能：让观众看到「如果主角没做X选择」的平行世界。
3. 对照差值 = 观众的爽感来源。
4. 主角赢了但不去踩对照组→展现大女主格局+保护温情基调。
5. 对照组的冲突强度不超过🔸🔸（可控级别）。

## 八、温情冲突分级

| 冲突类型 | 强度 | 解决方式 |
|---------|------|---------|
| 物质困境（没粮/没水） | 🔸🔸 | 主角能力解决 |
| 家庭内部（婆母最初抱怨） | 🔸 | 时间证明+温情 |
| 邻里对比（对照组酸话） | 🔸🔸 | 实力打脸+女主格局 |
| 外部阻力（村长质疑） | 🔸 | 用结果说话 |
| 天灾/敌人 | 🔸🔸🔸 | 全家协力 |

**前10集冲突不超过🔸🔸级别，** 先建立治愈感再逐步升高压力。

---

## 九、感情线推进规则

感情线不单独推进，必须**绑定在种田/经济/冲突事件上**：

```
阶段1（第1-5集）：种菜成功→他吃饭→他认可（被动接受）
阶段2（第6-10集）：上山遇险→他保护她→信任建立（外因互救）
阶段3（第11-15集）：荒地开荒→他公开站台→感情升级（公开表态）
阶段4（第16-20集）：赚钱给他买东西→他接受→双向奔赴（主动给予）
```

**"在一起做事情"中自然产生感情，不是靠甜言蜜语。**

## 十、视角切换规则

每集隐形视角分配：

```
种田/创业场景 → 女主视角（90%时间）
打架/护妻场景 → 男主视角（5%时间）
家庭/温情场景 → 婆母/旁观视角（5%时间）
```

- 女主POV → 观众获得「掌控感」
- 男主POV → 观众获得「安全感」（他在暗中关注/保护她）
- 第三者POV → 观众获得「成就感」（她已被认可）
