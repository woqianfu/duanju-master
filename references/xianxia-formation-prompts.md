## 五、与短剧大师各大师的对接

| 大师 | 受益方式 |
|:----|:--------|
| **武术大师** 🥋 | 16种阵法 → 对应16种武打场景分级（L1-L3），可直接调用对应提示词 |
| **运镜大师** 🎥 | 样片给出完整5秒分镜（推近→拉高→环绕→后拉），可迁移到其他阵法 |
| **灯光大师** 💡 | 每个阵法的专用配色 + 色光方向（八卦金/五行五色/雷阵紫黑等） |
| **构图大师** 🖼️ | 阵法的空间透视指引（水平铺展 vs 垂直光柱 vs 环绕结构） |
| **废片大师** 🗑️ | 通用负面词直接复用；"不要错误手指"在结印场景中尤其关键 |
| **矩阵大师** 📊 | 20维质量矩阵中新增「阵法结构清晰度」「结印手势准确」维度 |
| **省钱大师** 💰 | 结构化提示词 = 一次过片率高 = 废片率↓ = 直接省钱 |
