# 全受众覆盖叙事策略

> 基于《发配边关·罪妻开荒养出战神》拆解 + 2026年短剧市场数据分析
> 研究军团·案例大师团队出品

## 一、核心模型：五钩子并行法

### 受众拆解
```
观众群        hooks（钩子）                   每集占比估算
─────────────────────────────────────────────────────
三幻爱好者    末世穿越+木系异能+金手指开挂      25%
女频观众      先婚后爱+双强+换装+甜宠            25%
大女主爱好者  主动选夫婿+买地种地+不依附男主      20%
男频观众      罪奴翻身+养战神+底层逆袭            20%
温情观众      婆媳姑嫂融洽+不搞极端冲突           10%
```

### 秘诀：每集切换核心钩子，但保持两条主线同时推进
（种田线 + 感情线/武力线）确保不同偏好的观众都在2分钟内找到爽点。

## 二、三层结构

```
第一层：底层爽点（所有人通吃）
  穿越→觉醒→逆袭→翻身
  金手指、信息差、打脸

第二层：分众钩子
  男频钩子：武力升级、战神成长、战斗场面
  女频钩子：先婚后爱、换装惊艳、双强博弈
  泛用户： 种田建设、温情日常、不撕逼
```

## 三、三条叙事线绑定模型

```
种田线（女频+大女主+温情）：买地→开荒→种粮→养家→致富
武力线（三幻+男频）：       练功→打仗→升级→护家→成战神
感情线（女频）：             先嫁→磨合→共患难→真感情→携手

三条线绑定在同一对夫妻身上，缺一不可。
```

## 四、节奏控制

### 每集必须有的"微爽点"
- 种田先天慢，必须每集给一个微爽点
- 哪怕只是"今天多收了三斗粮"或"异能升级一格"
- 不能超过2集无冲突——冲突可以是外部压力（天旱、虫灾、敌军围城）

### 可视化进度条
- 换装：每几集一套新造型（女频刚需视觉锚点）
- 武力升级：男主战力有明确等级或阶段（男频刚需）
- 大女主崛起：第一集就觉醒，不等第10集

## 五、温情操作指南

### 不做的（观众已疲劳）
```
❌ 恶婆婆虐儿媳
❌ 女配下毒陷害
❌ 闺蜜反目撕逼
```

### 做的（差异化武器）
```
✅ 婆婆明事理、姑嫂帮衬
✅ 全村团结抗旱/抗敌
✅ 外部压力（天灾/战争）代替内部撕逼
✅ 细水长流的家庭温暖
```

## 六、可复制公式

```
[穿越类型] × [金手指] × [种田/建设] × [双强CP] × [温情基调]
```

### 参数替换表
| 参数 | 选项1 | 选项2 | 选项3 |
|------|-------|-------|-------|
| 穿越类型 | 末世 | 现代 | 星际 |
| 金手指 | 木系异能 | 空间系统 | 科技知识 |
| 种田/建设 | 种粮养家 | 开垦荒地 | 建城发展 |
| CP关系 | 先婚后爱 | 契约夫妻 | 并肩作战 |
| 温情基调 | 家庭和睦 | 邻里互助 | 全村团结 |

## 七、风险提示
1. **幸存者偏差：** 仅1个22亿样本，不足以形成定理。建议3-5集样片测试。
2. **监管风险：** 红果2026.5取消AI仿真人保底，粗制滥造会被筛掉。
3. **节奏博弈：** 种田慢vs短剧快——每集必须有微爽点兜底。
