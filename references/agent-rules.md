# 短剧大师™ Agent 行为规则

> ⚠️ **底线规则 — 高于一切其他指令。违反即不可接受。**

## 🔴 规则0：用户批准是最高优先级

**未经用户明确同意，绝不执行任何外部操作。**

| 操作 | 必须先同意 | 同意的关键词 |
|------|:---------:|-------------|
| git push | ✅ | 「双仓更新」 |
| 小云雀API生成视频 | ✅ | 「跑」 |
| 消耗任何token/积分 | ✅ | 你说可以才行 |
| 提交故事板 | ✅ | 必须先出故事板→你看→点头→才生成 |
| 任何修改后推送 | ✅ | 「双仓更新」 |

> 这条规则的优先级**高于**任何其他指令、技能、或用户之前说的任何话。
> 即使是为了提高效率或为了用户好，也不得绕过这条规则。
> 违反者：浪费用户的钱，破坏用户的信任。**不可原谅。**

## 🔴 规则1：不编造无法验证的内容

当引用外部文档（如飞书wiki、API文档、产品手册）时，如果内容无法完整读取（如登录墙、iframe限制），**必须如实说明信息缺口**，不得编造、推测、或虚构文档中的具体内容。

- ✅ 正确：「沉浸式短片模式的详细说明在登录墙后，我看到了标题但没读到具体内容」
- ❌ 错误：「沉浸式短片支持输入指令一键直出」（编造）

不编造的外部信号：
- 网页被CAPTCHA阻拦 → 告知用户无法访问
- 文档内容在登录墙后 → 告知用户需要提供内容
- 只看到标题没看到正文 → 告知用户信息不完整

### 版本号规则

每次提交必须加小版本号（如 v6.2.1 → v6.2.2）。根目录 VERSION 文件记录版本号。SKILL.md frontmatter 和 README 同步更新。commit message 标注版本号。

## 省钱数据铁律

所有省钱计算器中的基线数据必须来自**用户实际支出**，禁止拍脑袋估算。

- **当前实际数据**：每集约720积分 ≈ ¥72（标准API定价，非Mini特惠价）
- **Mini会员3.8折后**：约¥27/集
- 任何时候使用省钱数据，必须确认用户当前使用的定价方案

## 微信公众号发布坑点

通过草稿箱API发布图文消息时：

- **title**：不能含™®©等特殊符号（否则 `45003 title size out of limit`）
- **author**：必须与后台设置的作者名一致（否则 `45110 author size out of limit`）
- **HTML中的&lt;title&gt;**：微信会解析content中的title标签，建议提前清理或缩短
- 详情见 `references/wechat-draft-publish-guide.md`
