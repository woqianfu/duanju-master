# 剧本输出格式规范

适用于短剧大师技能输出剧本时的排版规则。

## 核心原则

不要用任何 Markdown 装饰符号。剧本就是剧本，不是文档。

## 禁用符号清单

| 符号 | 替代方案 |
|------|---------|
| `**` 加粗 | 直接写文字，不加粗 |
| `---` 分割线 | 不保留，靠空行分隔 |
| `###` 标题 | 直接写标题文本 |
| `>` 引用 | 直接写内容 |
| `（竖屏：xxx）` | 直接写 xxx，去掉括号和"竖屏"前缀 |
| `「」` | 直接写内容，去掉装饰性书名号 |
| `*` 斜体 | 直接写文字 |
| `（接上）` / `（接第X集）` | 去掉，场景标题自包含 |

## 标准剧本段落格式

```
场景编号. 地点 内景/外景 时间

动作描述段落...

角色名
（动作语气）
台词内容。
```

### 规则

1. **场景标题**：`数字. 地点 内景/外景 时间`，自包含，不加括号
   - ✅ `1. 莫高窟第61窟 内景 日`
   - ❌ `1. 莫高窟第61窟 内景 日（竖屏：窄窄的洞窟甬道）`

2. **动作描述**：纯文本段落，不加任何标记

3. **角色名**：独占一行，不加 `**` 不加冒号
   - ✅ `林小禾`
   - ❌ `**林小禾**`

4. **动作语气**：跟在角色名下一行，用圆括号包裹
   - ✅ `林小禾` 换行 `（近乎耳语）` 换行 `一模一样。`

5. **台词**：角色名/动作语气行之后，直接写台词

6. **画外音**：用 `(OS)` 标注，如 `林小禾(OS)`

7. **片头/片尾**：用 `【片头】` `【片尾】` 标注

8. **下集预告**：直接写"下集预告字幕："后跟内容，不加 `*` 斜体

9. **场景过渡**：场景之间用空行分隔，不用 `---`

## 整体文件结构

```
《剧名》

参赛方向：xxxx

一句话介绍
（内容）

故事大纲
（内容）

人物小传

角色名
（描述）

角色名
（描述）

第1集：集名

【片头】
片头描述...

1. 地点 内景/外景 时间
动作描述...

角色名
台词。

【片尾】
片尾描述...

第2集：集名
...
```
