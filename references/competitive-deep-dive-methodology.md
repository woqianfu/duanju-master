# 竞品深度拉片方法（案例大师核心方法论）

## 适用场景
需要对一部爆款短剧做竞品拆解时，从公开数据重建其剧集结构、节奏规律和创作公式。

## 数据来源优先级
1. **番茄小说/原著地址**——短剧大多有原著小说，章节标题透露剧情结构
2. **YouTube合集描述**——"一口气看完"类视频对剧情有完整概括
3. **Facebook帖子/评论区**——用户讨论揭示真实的受众共鸣点
4. **抖音/红果评论区**——高赞评论揭示观众真正在意的爽点
5. **搜索剧名+关键词组合**——"第一集 剧情" "开局" "名场面" "大结局"

## 五步拉片法

### 第一步：收集骨架数据
```
搜索："<剧名> 剧情 第一集 开局 穿越 介绍" → 获取开局设定
搜索："<剧名> 小说 章节" → 获取章节目录（映射剧集）
搜索："<剧名> 评论" → 获取观众真实反馈
```

### 第二步：利用小说章节目录映射剧集结构
小说184章 ÷ 剧集66集 ≈ 2.8章/集
按此比例推算每集覆盖的章节范围，再从章节标题反推剧集内容。

### 第三步：逐集重建内容（从三类线索）
| 线索类型 | 来源 | 能重建的程度 |
|---------|------|------------|
| 小说原文 | 原著章节内容 | 对话+场景可还原80% |
| YouTube标题+描述 | 视频简介 | 关键事件点可还原 |
| 合辑长描述 | "一口气看完"标题 | 完整剧情线可还原 |

### 第四步：七维分析框架
对每集/每板块分析：
1. **种田线**——经济/建设进展
2. **感情线**——CP关系进度
3. **武力线**——男主成长
4. **对照组线**——对比反派的生命周期
5. **冲突/危机线**——外部压力级别
6. **社会地位线**——女主在社群中的位置
7. **能力线**——金手指的进化

### 第五步：提取可复制规则
从拆解中总结：
- 每几集一个闭环
- 每个闭合的情感锚点是什么
- 对照组的生命周期
- 危机升级的阶梯
- 三线是否锁扣

## 拉片报告标准结构
```markdown
## 第X集「标题」

### 内容逐帧重建
| 时间线 | 场景 | 内容 | 叙事功能 |
|--------|------|------|---------|

### Hook拆解
- 第N个反转（时间点）：描述
- 核心钩子：一句话

### 受众覆盖
- 🏆 三幻：xxx
- 💑 女频：xxx
```

## 关键陷阱
1. ❌ 小说 ≠ 剧集一一对应——漫剧会删减支线，只保留主线冲突
2. ❌ YouTube合集的标题可能夸张——交叉验证
3. ✅ 原著小说的标签(如"对照组""种田""甜宠")是最可靠的索引
4. ✅ Facebook群组的真实观众讨论是最真实的数据

## 用户偏好笔记
- 男主终局称号要有"天下级"的压迫感，如"战神"＞"猎王"、"大将军"＞"统领"
- 标题设计原则：让观众第一眼就知道这是"最顶级"的设定
