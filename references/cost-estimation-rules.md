# 省钱计算实战指南 — 成本优化红线

在短剧大师省钱场景中，成本估算必须基于**真实用户数据**。

## 🔴 死规则：永不捏造成本数字

**历史教训**：在省钱计算器中写了"450元/60集"，但这是拍脑袋的估算。用户实际是¥72/集×60=¥4,320，差了近10倍。

**正确的做法**：
1. 涉及"当前花了多少钱"——**先问用户，不要自己算**
2. 用户给了数字（如「720积分≈72元」）→ 用这个数字作为基线
3. 所有省钱比例按这个基线百分比计算，而不是自创一个数字
4. 在文档中明确标注「基于用户实际数据」

## 用户真实基线（参考）

| 项目 | 数据 |
|------|------|
| 当前每集 | ~720积分 ≈ ¥72 |
| 60集 | ¥4,320 |
| 定价类型 | 标准定价（未用Mini会员折扣） |
| Mini会员3.8折价 | ~¥27/集 |
| 终极预估 | ¥10-18/集（省80-85%） |

## 误区排查

当用户说"不对"时，立刻停止估算并问实际数据。不要试图修正一个自创的数字——它从一开始就是错的。
