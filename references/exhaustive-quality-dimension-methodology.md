# 穷举法AI视频质量全维度控制方法论

> 2026-06-14 | 本会话产出 | 短剧大师技能参考文件

## 核心思路

AI视频生成质量控制不能靠"感觉哪里不对"——必须穷举一切可能出现问题的维度，每个维度建立五列闭环：

```
一句话标准 → 控制方法(prompt级) → 负面约束(英文锁定) → 检测手段(量化阈值) → 修复方案(递进重试)
```

## 20维度分类法

| 分类 | 维度数 | 范围 |
|------|:--:|------|
| 画面一致性 | 12 | 面部/服装/场景/道具/光照/色调/运动/运镜/景深/比例/特效/背景 |
| 声画同步 | 4 | 口型(≤0.5s)/表情/BGM/字幕 |
| 技术质量 | 3 | ≥30fps/伪影检测/物理合理性 |
| 叙事逻辑 | 1 | 镜头序列=剧本事件序列 |

## 方法论复用指南

1. **先列维度**：任何新场景/新平台，先穷举可能出问题的维度
2. **再定阈值**：每个维度的通过标准必须量化（≤偏差值/≥最小值）
3. **后建修复链**：从轻→重的递进修复策略（改prompt→换参考图→重抽→人工）
4. **最后做总表**：提交前20维快速扫描（附录R.21模板可复用）
