# 短剧大师™ 软著/DCI申请指南

> 基于2026-06-15实战申请经验。平台：版灯 ccDCI.cn

## 申请流程

### 1. 准备材料

| 材料 | 要求 |
|------|------|
| 作品名称 | 纯中文+英文，禁止括号/符号。如「短剧大师微短剧创作系统规则引擎」 |
| 作品类别 | 文字作品（Skill的SKILL.md+全部references视为一个文字作品整体） |
| 申请人 | 著作权人全称（如：北京创业科创科技中心） |
| 作者 | 同著作权人或实际创作者 |
| 创作完成日期 | 首次完整版本日期 |
| 首次发表日期 | GitHub首次公开push日期 |

### 2. 作品内容准备

- **PDF格式**：文本型PDF（可检索），非扫描件
- **内容要求**：纯文字，无代码块、无表格、无特殊符号、无Markdown标记
- **命名规则**：不含括号/空格/特殊字符
- **大小限制**：建议<10MB

### 3. 提交要点

1. 登录 ccDCI.cn（版灯平台）
2. 填写作品信息（名称须与PDF内标题完全一致）
3. 上传文本型PDF
4. 确认哈希值自动生成
5. 等待审核（通常即时出证）

### 4. 获证后操作

- 下载PDF证书 → 存入 `assets/DCI-数字版权证书.pdf`
- 在SKILL.md的品牌标识区填入DCI编号
- 更新商标输出铁律，将DCI嵌入品牌尾标
- `git commit` + 双仓库推送

## 常见拒绝原因

| 原因 | 修复 |
|------|------|
| 名称含括号/符号 | 去掉所有符号，只保留中英文 |
| PDF非文本型 | 重新导出为可检索文本PDF |
| 内容含代码/表格 | 精简为纯文字描述 |
| 名称与PDF内标题不一致 | 确保完全一致 |
