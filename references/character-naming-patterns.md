## 六、角色名选配流程

```
「命名大师，为[角色名]起配套角色名字」
```

### 6.1 主角团命名

围绕核心姓氏组织家族式命名，保持统一感：

| 角色 | 人名 | 命名逻辑 |
|:----|:-----|:---------|
| 男主角 | 钱有才 | 才华横溢，名字暗示逆袭潜质 |
| 女主角 | 钱千金 | 大家闺秀身份锁定 |
| 反派 | 钱亿万 | 有钱即有权，名字暗示财阀背景 |
| 师父 | 钱真人 | 世外高人，道风仙骨 |
| 忠诚管家 | 钱守财 | 名字暗示职业和忠诚 |

### 6.2 风格匹配对照

| 短剧风格 | 推荐命名模式 | 举例 |
|:---------|:------------|:-----|
| 🏔️ 仙侠 | 辈分尊称 + 身份大号 | 钱父/钱真人/钱道长 |
| 💼 都市霸总 | 富贵命格 + 身份大号 | 钱亿万/钱老板/钱先生 |
| 🤖 科幻 | 气质型 + 意境词 | 钱无限/钱无尽/钱无量 |
| 🤣 喜剧 | 富贵命格 + 有X系列 | 钱满贯/钱有喜/钱来也 |
| 🎭 古装权谋 | 身份大号 + 意境词 | 钱员外/钱九鼎/钱永康 |
| 🌆 民国 | 有X系列 + 身份大号 | 钱有为/钱先生/钱老爷 |

## 七、命名大师完整工作流

```
1. 确认姓氏：用户指定 or 默认"钱"姓 or 自动匹配题材常见姓
2. 选命名模式：按题材 → 查6.2风格匹配对照
3. 输出5个候选名 + 人设关键词
4. 用户选定 → 写入角色档案
5. 全剧保持该角色名一致性
```

> **命名原则**：短剧角色名要**三个字以内**（含姓）最朗朗上口。四个字可用但需有意境。
> **配套联动**：选好名字 + 查发型库 + 查服装库 = 完整角色包
