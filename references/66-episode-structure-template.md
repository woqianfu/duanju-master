# 66集短剧结构模板（基于爆款拉片提取）

## 适用场景
从零规划一部S1共66集的短剧时，按照此模板设计节奏。

## 核心结构：六大板块

```
板块1 (1-5集)   定锚期
板块2 (6-10集)  上升期
板块3 (11-20集) 爆发+考验期
板块4 (21-35集) 扩张期
板块5 (36-50集) 新能力/升级期
板块6 (51-66集) 高潮+收尾期
```

## 详细模板

### 板块1：定锚期（第1-5集）
**任务：** 穿越定锚·关系定锚·困境定锚·金手指展示·第一次正反馈
**必打标点：**
- 第1集 hook（选人/选环境时的反常规决策）
- 第3集前亮金手指（不能藏太久）
- 第5集第一次正反馈（有第一笔收入/第一口饱饭）
- 男主最初冷淡态度（为感情弧线铺路）

### 板块2：上升期（第6-10集）
**任务：** 拓展资源·对照组上线·开荒扩产·第一次小冲突·感情升温
**必打标点：**
- 第7-8集对照组上线
- 第10集第一个感情里程碑（"以后这就是咱家了"级别）
- 第一次外部小冲突（用实力/智慧化解）

### 板块3：爆发+考验期（第11-20集）
**任务：** 对照组深化→闭环·第一次天灾/外部危机·感情质变·经济翻升
**必打标点：**
- 对照组走完完整生命周期（嫉妒→造谣→被打脸→服气→退出，约在第16集前）
- 第17-20集第一次天灾危机
- 第20集感情质变（"还好有你在"级别）

### 板块4：扩张期（第21-35集）
**任务：** 个人富→带全村富·多元化经济·基建改造·声望建立
**必打标点：**
- 第24-26集第一次大丰收/大成果
- 第30集前从"个人"扩展为"带领群体"
- 第33-35集回顾收尾（回顾前半段的成长）

### 板块5：新能力/升级期（第36-50集）
**任务：** 引入第二能力·男主上升线启动·外部威胁预埋
**必打标点：**
- 第36-38集新能力/新伙伴引入（避免种田疲劳）
- 男主在团队中的领导力第一次被外界验证
- 第48-50集建立板块6的对手/威胁

### 板块6：高潮+收尾期（第51-66集）
**任务：** 最大危机·双强合体·终局成长·首尾呼应
**必打标点：**
- 三条线在此交汇（经济、感情、武力都在同一场决战中用到）
- 第66集首尾呼应（第1集的某个元素回到结尾画面）
- 所有伏笔回收

## 三线锁扣规则

```
女主经济/能力线 → 男主武力/成长线
       ↓                ↑
   互相提供资源 ← 互相保护
```

**感情节点永远绑定事件节点：**
事件正反馈 → 男主内心变化 → 一个沉默但有力的行动 → 感情前进一格
不单独做"谈情说爱"的戏，感情在日常奋斗中自然生长。

## 对照组生命周期

```
上线(第7-8集) → 嫉妒(第11-12集) → 造谣(第12-14集)
→ 被打脸(第14-16集) → 服气/退出(第16集)
```
对照组走完弧线后必须退出，不要重复使用。更大的冲突留给外部威胁。

## 危机升级阶梯

| 阶段 | 强度 | 位置 | 解决方式 |
|------|------|------|---------|
| 物质困境 | 🔸 | 板块1 | 金手指解决 |
| 他人眼光 | 🔸🔸 | 板块2 | 实力自证 |
| 对照组/轻度冲突 | 🔸🔸🔸 | 板块3前段 | 实力+善良 |
| 第一次天灾 | 🔸🔸🔸🔸 | 板块3后段 | 预判+团队 |
| 外部恶势力 | 🔸🔸🔸🔸🔸 | 板块5后段 | 男主武力 |
| 最大天灾/决战 | 🔸🔸🔸🔸🔸🔸 | 板块6 | 双强合体 |

## 首尾呼应设计

| 维度 | 开头 | 结尾 |
|------|------|------|
| 地点 | 破败→ | 繁荣 |
| 关系 | 陌生人→ | 生死相依 |
| 男主 | 被看低→ | 封号 |
| 女主 | 罪臣之女→ | 受人敬重 |
| 经济 | 家徒四壁→ | 丰衣足食 |

## 男主终局称号设计原则
- 终局称号必须有"天下级"压迫感：**战神/大将军/侯爷/箭神** > 村王/猎王/好汉
- 称号要让观众第一眼就感受到"这人是世界顶级的"
- 前期可以低调（猎户/百户/斥候），但终局必须拉满
