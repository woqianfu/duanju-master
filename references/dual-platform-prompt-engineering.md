# 双平台统一提示词工程方法

> 2026-06-14 | 本会话产出 | 短剧大师技能参考文件

## 适用范围

当技能需要同时支持多个AI视频生成平台（小云雀/Seedance 2.0/其他）时，使用统一公式+平台参数映射的方法，避免为每个平台维护独立的prompt系统。

## 统一6步公式

```
[主体] + [动作] + [场景环境] + [运镜] + [风格光影] + [负向约束]
```

## 平台参数映射表模板

| 维度 | 平台A | 平台B | 统一策略 |
|------|------|------|------|
| 提示词上限 | N字 | M词 | 取min(N, M×3) |
| 视频时长 | A-Bs | C-Ds | 统一范围 |
| 分辨率 | X | Y | 取交集或性价比最优 |
| 输入模态 | ... | ... | 用交集能力 |
| 特色能力 | ... | ... | 按场景分配 |

## 关键原则

1. **一公式双平台**：prompt主体结构不变，只在参数层做平台适配
2. **字数预算隔离**：各平台独立预算，取最小值确保双平台通用
3. **负向约束必须**：即使平台不支持独立负面词字段，也要在prompt中内嵌英文负面关键词
4. **特色能力按需分配**：对话→音频平台，武打→动态平台，固定→稳定性优先
