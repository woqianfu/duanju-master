## 核心公式

```
【核心对抗主体 20%】+ 【物理受力与动作状态 30%】+ 【环境场景 15%】+ 【分镜运镜 20%】+ 【视觉风格 15%】
```

各维度权重表示描述词分配比例——越高的维度在prompt中占据越多字数。

## 维度详解

### 1️⃣ 核心对抗主体 (20%)

明确交战双方的**身份、体型、装备、能力**四大要素。差异越大，AI生成的碰撞感和戏剧冲突越强。

**避坑**：避免"两人打架""剑客对决"等模糊描述 → AI生成同质化动作。

**好例**: "机械义体改造人(右臂为电磁拳套) vs 古风炼气士(手持桃木剑，周身淡青灵气)"

### 2️⃣ 物理受力与动作状态 (30%) — 决定是否"假打"的关键

用**"受力"替代"动作"**——不是写"快速出拳"，而是写发力前的蓄力、动作中的受力、击中后的反馈、身体状态。

| 阶段 | 描述要点 | 提示词示例 |
|------|---------|-----------|
| 蓄力 | 重心/肌肉/拳头位置 | "双腿分开与肩同宽，膝盖微屈压低重心，手臂肌肉虬结绷紧" |
| 受力 | 防御/被击反作用 | "左臂硬接对手重拳，身体因冲击力向后滑出半米" |
| 反馈 | 受伤/消耗/硬直 | "铠甲凹陷碎裂，嘴角溢出血丝，左手按住受伤部位" |
| 状态 | 疲劳/持续战斗 | "肌肉紧绷颤抖，胸口剧烈起伏，额头汗珠滴落" |

### 3️⃣ 环境场景 (15%)

场景不是背景——是放大力量感的载体。描述**战斗如何改变环境、环境如何影响战斗**。

- 巷战：拳脚冲击力震落墙上广告牌，积水被气浪掀起
- 竹林战：气劲斩断竹枝，竹叶漫天飞舞
- 废墟战：战斧劈在水泥柱上，水泥块剥落飞溅

### 4️⃣ 分镜构图与运镜 (20%)

**张弛有度**——用停顿/推进/慢放/切镜营造蓄力和冲击力。

- 近战：特写聚焦拳头(蓄力) → 命中瞬间快速推进+微震 → 慢动作近景
- 刀剑：低角度仰拍全景 → 特写刀刃碰撞火花 → 环绕运镜
- 巨型对决：史诗广角远景 → 快速推拉镜头聚焦爪尖 → 震动镜头

### 5️⃣ 视觉风格 (15%)

明确画风/渲染/色彩/细节精度。**避免宽泛描述**。

| 风格 | 关键词 |
|------|-------|
| 2D热血日漫 | 赛璐璐平涂、高饱和、速度线、残影、黑白闪 |
| 国风水墨武侠 | 硬边水墨、低饱和墨色、留白构图、朱砂红点缀 |
| 赛博条漫 | 厚涂、青紫基调、霓虹反光、金属质感 |
| 3D史诗CG | 虚幻引擎5、体积光、物理破坏、8K材质纹理 |
| **机甲科幻** 🆕 | 硬表面建模、金属碰撞火花、蓝光能量刃、机甲装甲碎裂 |

## 实战三步法

### 第一步：写"对抗"而非"动作"

用冲突逻辑替代动作指令——先描述局势(谁压制谁)，再描述角色如何发力/反击。

### 第二步：补充重量感与疲态

让角色有血有肉——落地有闷响、战斗会喘、拳头要青筋暴起。

### 第三步：镜头节奏放大力道

快慢结合——蓄力时镜头停顿0.5s，命中时快速推镜+震动，命中后切慢动作。
