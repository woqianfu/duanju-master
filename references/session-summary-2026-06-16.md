# 2026-06-16 会话摘要

## 核心教训：数据铁律

省钱计算器中的数字必须基于用户实际数据。本会话中用户说：
- 当前每集成本：720积分 ≈ 72元（标准定价，非Mini价）
- 60集总成本：4,320元（不是之前拍脑袋的450元）

**修正后的省钱路线：**
```
当前：72元/集，60集=4,320元
Mini会员3.8折 → 27元/集 → 60集1,620元
+ 镜长65s→60s(-8%) → 约1,490元
+ 降废片60%→15% → 约820元
+ Seed固定+24fps → 约680元
理论上限：10-18元/集
```

## 发布的策略优化

## 关键决策

- author字段必须是公众号已注册的「作者名」，不是任意字符串
- 图片URL必须每次重新上传获取，media_id不能直接当URL用
- 发布前必须用curl验证所有图片URL返回200
- HTML中特殊字符(™/¥/emoji)会被微信接口转成unicode，必须替换
- 中文用json.dumps(ensure_ascii=False)确保不被转义
