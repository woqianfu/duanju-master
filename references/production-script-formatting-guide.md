# 拍摄级剧本格式指南

> 2026-06-27 从《代码里的长安》60集实战中提炼
> 短剧大师v6.2.4 · 研究军团产出

## 一、为什么需要拍摄级格式

60集长线大纲只解决「讲什么故事」的问题。拍摄级剧本解决「怎么拍」的问题。

| 维度 | 大纲级 | 拍摄级 |
|:----|:-------|:-------|
| 每集内容量 | 1-3段描述 | **8-11个镜头** |
| 时间控制 | 无 | **逐镜标注时间码 0-62s** |
| 场景过渡 | 隐含 | **旁白/画外音显式连接** |
| 拍摄指导 | 无 | **环境/角色状态/镜头提示** |
| 单集时长 | 约15-30秒 | **约55-65秒（达标）** |

**核心原则：** 每一集单独拿出来，去掉上下文，导演拿着就能拍。不能依赖「前面说了」的信息。

## 二、五段式逐秒结构

每集60秒必须按以下五段分配：

```
00:00 ───── 05 ───── 15 ───── 35 ───── 50 ───── 62
  ↑          ↑          ↑          ↑          ↑
 黄金钩子   情境定位   推进/冲突   高潮点      悬念收尾
 (0-5s)     (5-15s)    (15-35s)   (35-50s)   (50-62s)
```

### 段落功能说明

| 段落 | 时长 | 功能 | 禁止做的事 |
|------|:----:|------|-----------|
| 黄金钩子 | 0-5s | 第一帧抓眼球，让拇指停下来 | 上集回顾、风景空镜、旁白铺垫 |
| 情境定位 | 5-15s | 快速交代谁在哪在干嘛 | 大段回忆、角色自报家门 |
| 推进/冲突 | 15-35s | 核心事件，情绪爬升 | 无关插科打诨、信息灌水 |
| 高潮点 | 35-50s | 本集最燃/最暖/最炸瞬间 | 提前泄高潮、高潮后附加解释 |
| 悬念收尾 | 50-62s | 必须看下一集的理由 | 请看下集、虚假悬念 |

## 三、每镜必备要素

每个镜头必须包含以下内容（按优先级）：

### 1. 场景描写（不可省略）
```
**镜3** 莫高窟夜色。月牙悬在九层楼上空。61窟铁门被推开，吱呀声在走廊里回荡。
```

**好的描写**包含感官细节：
- 视觉：光线、颜色、动作
- 听觉：环境音、背景音
- 嗅觉/触觉（密室场景特有）：灰尘、墨味、腐朽木香、干燥

**差的描写**只有抽象概括：
- ❌ 林依依进入一个神秘的密室
- ✅ 手电筒光扫过墙壁，灰尘在光影中飞舞。空气中弥漫着干透的墨味和腐朽的木香。

### 2. 旁白/画外音（用于过渡和上下文）
```
（旁白）一千二百七十一年前——有人在敦煌的石壁上藏了一个秘密。
```

**旁白的功能：**
- 连接两个时空的跳跃（回忆与现实之间）
- 提供观众需要但角色不需要明说的背景信息
- 制造节奏感——在紧张段落插入旁白让观众"喘口气"
- 出现在观众可能会困惑的时刻

**频率：** 每集至少3段旁白，最多不超过6段。每段不超过20字。

### 3. 角色+对白
```
**林依依**：北纬40度02分……东经94度48分……鸣沙山顶？
```

**对白规则：**
- 单句不超过25字
- 保留口语感、潜台词
- 不通过对白解释设定（「其实我是XX的后代」）
- 不说话的角色行为更有力量

### 4. 时间码（每镜必备）
```
**镜1** **0-5s** 内容描述
```

时间码精确到秒级，供导演和后期逐镜卡时长。

## 四、60集节奏控制的实战经验

### 4.1 情感点分布

60集中情感高点不能全堆在一段。平均分布法则：

| 段落 | 集数 | 情感点数量 | 类型 |
|:----|:----:|:----------:|:-----|
| 建立期 | E01-E10 | 4-5个 | 🔥燃为主 |
| 展开期 | E11-E22 | 4-5个 | 💧泪+🔥燃交替 |
| 升级期 | E23-E38 | 5-6个 | 💧泪为主 |
| 转折期 | E39-E52 | 4-5个 | 混合 |
| 终局期 | E53-E60 | 4-5个 | 全释放 |

**关键发现：** E01-E12情感点过少（仅有4个）会导致观众流失。前12集至少安排6个情感高点，每2集至少1个。

### 4.2 尾钩质量分级

| 级别 | 特征 | 示例 | 追更动力 |
|:----:|:-----|:-----|:--------:|
| S级 | 一句话颠覆认知 | 「查一下她外婆的死因」 | 极强 |
| A级 | 新威胁出现/身份揭露 | 「笛子背面刻着一个连十七都不知道的名字」 | 强 |
| B级 | 信息悬念 | 「长安最美的地方是哪里？十七也想不起来了」 | 中等 |
| C级 | 情绪延宕 | 「这句话十七记了一千二百年」 | 弱（需强化） |

**60集尾钩分布规则：** 至少30%为S/A级，最多20%为C级。C级尾钩必须通过改写强化为B级以上。

### 4.3 暗线覆盖密度

多条暗线（如外婆线、双向解码线、组织线）必须保持持续存在感：

| 暗线长度 | 最大中断集数 | 穿插方式 |
|:---------|:-----------:|:---------|
| 贯穿全剧 | 不超过10集 | 每10集至少1次明确提示 |
| 中段暗线 | 不超过5集 | 每5集至少1次暗示 |
| 背景暗线 | 不超过15集 | 每15集至少1次伏笔回收 |

**实战教训：** 外婆线在E05埋设后中断了16集（到E21才重新出现），导致观众忘记这条线。最佳做法是E09/E10/E13各穿插1个极短镜头（1-2句）。

## 五、60集写作流程

### Step 1: 先写大纲
- 确定五段结构（建立→展开→升级→转折→终局）
- 每段核心任务和反派梯度
- 主要角色弧线（每个角色至少3个转折点）

### Step 2: 再写分集概要
- 每集1-3句话描述核心事件
- 标注该集的情感类型（🔥💧🤔）
- 标注该集的暗线提示点

### Step 3: 扩写为拍摄级剧本
- 按五段结构分配8-11个镜头
- 每镜标注时间码
- 添加旁白做过渡
- 确保每集内容量足够60秒

### Step 4: 自检清单
- [ ] 每集是否至少有8个镜头？
- [ ] 每集是否包含3-5段旁白？
- [ ] 前3秒是否有抓人的画面或冲突？
- [ ] 15秒内观众能知道这是哪谁在干嘛？
- [ ] 高潮点是不是本集最燃/最暖/最炸的瞬间？
- [ ] 50-60秒的钩子是不是真的让人想看下一集？
- [ ] 去掉所有台词光看画面能看懂剧情吗？
- [ ] 有没有任何「上集说到」或废话开头？
- [ ] 这一集单独拿出来好看吗？

**3个以上不过 → 重写该集。**

## 六、常见陷阱

### 陷阱1：一集只有1-2场戏
**表现：** 信息密度不足，30秒就说完了  
**修复：** 每集至少3场戏，5-8个镜头。把动作拆细——进门、环顾、特写、反应，每个动作拆成一个镜头

### 陷阱2：没有旁白过渡
**表现：** 场景切换生硬，观众不知道时间地点变了  
**修复：** 每集3-5段旁白。旁白的功能是「观众需要但角色不需要知道的信息」

### 陷阱3：情感点扎堆
**表现：** 10集内集中了6个泪点，前后20集只有3个  
**修复：** 把关键情感事件拆开，每2-3集安排1个高潮。不能让观众连续哭3集也不能让观众10集无感

### 陷阱4：演员不知道该怎么演
**表现：** 剧本只写「林依依很难过」，没有具体动作指引  
**修复：** 写具体的可视化的动作——「林依依的手指停在键盘上方」「她没有哭，只是把嘴唇抿成一条线」
