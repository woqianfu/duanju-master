## 概述

GPT-Image-2 可一键生成 3D 动漫人物三视图，配合结构化提示词实现稳定的角色设计。本文提炼其提示词模板、六维设计要素和风格规律，作为短剧大师「造型大师」的知识基础。

## 六维设计要素框架

### ① 脸型 + 五官

| 要素 | 常用值 | 变体 |
|------|--------|------|
| 脸型 | 鹅蛋脸 | 瓜子脸、圆脸 |
| 皮肤 | 白皙如凝脂/无瑕 | 健康小麦色 |
| 眼型 | 丹凤眼（狭长）、桃花眼（圆润含水）、柳叶眼 | 根据气质选 |
| 眉型 | 远山黛（细长柔美）、柳眉 | 剑眉（英气） |
| 鼻型 | 琼鼻秀挺、小巧挺翘 | 高挺 |
| 唇型 | 朱唇、饱满如樱花瓣、薄唇蔻丹色 | |

### ② 发型 + 发饰

| 要素 | 常用值 | 说明 |
|------|--------|------|
| 发型 | 灵蛇髻、双环望月髻、半束半披高髻 | 古典发髻 + 飘逸散发 |
| 刘海 | 龙须刘海（鬓边垂落一缕） | 几乎必选 |
| 鬓发 | 鬓发垂落耳畔/耳侧 | 增加灵动感 |
| 发饰 | 金步摇、珠花、玉簪、镶金白玉蝴蝶步摇、粉蓝珠串流苏、绒花 | 层次丰富 |

**规律**：所有角色都有「龙须刘海 + 鬓发垂落 + 多层级发饰」

### ③ 服饰结构

```
交领右衽 + 外罩广袖纱衣 + 内衬抹胸 + 腰带 + 裙 + 鞋

变体组合：
- 劲装款：交领右衽劲装 + 高开叉 + 修腿 + 短靴 → 干练性感
- 仙裙款：交领右衽广袖长裙 + 轻纱外罩 + 飘带 → 仙气飘逸
- 多层叠穿：多层领口（露出内衬）+ 渐变外衫 + 腰带玉佩流苏
```

### ④ 配色方案

| 主角色 | 辅角色 | 点缀色 |
|--------|--------|--------|
| 月白 | 靛蓝 | 金丝银线绣纹 |
| 月白/靛蓝渐变 | 鹅黄 | 玉佩/流苏 |
| 月白/靛蓝渐变 | — | 暗纹云纹+莲花 |

**规律**：月白+靛蓝是核心组合，鹅黄做渐变过渡。所有角色共用这套配色体系。

### ⑤ 材质渲染

```
丝绸光泽 + 纱质轻盈 + 暗纹刺绣 + 3D高精度渲染 + 光照柔和通透
```

- 丝绸体现质感
- 纱质体现仙气
- 暗纹（云纹/莲花）体现细节
- 高精度渲染体现3D品质

### ⑥ 气质人设

| 气质类型 | 适用角色 | 对应眼型 |
|----------|---------|---------|
| 温柔 | 善解人意型 | 桃花眼 |
| 清冷脱俗 | 高冷仙子型 | 丹凤眼 |
| 灵动可爱 | 少女型 | 桃花眼+微笑 |
| 高贵冷艳 | 权贵/高傲型 | 丹凤眼+气场 |
| 性感 | 御姐型 | 眼尾微挑 |

## 实战案例：4套角色设计解析

### 案例1：凌波仙·婉清（温柔型）

| 维度 | 设计 |
|------|------|
| 脸型五官 | 鹅蛋脸，皮肤白皙如凝脂，柳眉如烟，眼眸似含秋水，琼鼻秀挺，朱唇轻点 |

### 案例3：瑶光（灵动可爱型）

| 维度 | 设计 |

### 案例4：白凤瑶（高贵冷艳型）

| 维度 | 设计 |
|------|------|

| 服饰结构 | 交领右衽广袖长裙→外罩丝绸开衩裙→高开衩露腿→玉带→流苏环佩→银色雕花护腕→软缎绣鞋 |
| 材质渲染 | 材质渲染细腻，光照柔和通透，丝绸光泽+纱质轻盈 |
| 气质人设 | 高贵冷艳，身材高挑苗条 |

### 案例对比矩阵

| 要素 | 婉清（温柔） | 云姬（清冷） | 瑶光（灵动） | 白凤瑶（高贵） |
|------|------------|------------|------------|--------------|
| 眼型 | 含秋水 | 丹凤眼 | 桃花眼+琥珀瞳 | 丹凤眼 |
| 发型 | 双环望月髻 | 灵蛇髻 | 灵蛇髻 | 半束半披高髻 |
| 配色 | 月白+靛蓝 | 靛蓝+月白 | 月白+靛蓝+鹅黄 | 月白+靛蓝渐变 |

### 跨集人物一致性规则

1. **所有角色需建立「视觉档案」**：每角色一页，含正面/侧面/背面/面部特写
2. **跨集复用**：同一角色每次出现，锁定相同Seed+相同Prompt模板
3. **换装原则**：每3-5集可换一套新造型（女频刚需），但核心辨识元素（发型/脸型/气质）不变
4. **服饰色彩体系**：短剧应建立自己的品牌色（类似月白+靛蓝），所有角色共用

## 对造型大师的补充建议

1. **现已加入短剧大师#22**：造型大师已正式列为第22位大师，隶属技术军团
2. **与肌肉大师的分工**：肌肉大师管面部表情动态（六维表情精控），造型大师管静态设计（脸型五官+发型+服饰）
3. **与构图大师的分工**：构图大师管镜头画面布局（竖屏七法/八站位），造型大师管角色本身视觉
4. **与剧本大师的衔接**：剧本大师输出角色描述（年龄/身份/气质）→ 造型大师输入 → 输出结构化提示词 → 交付给 GPT-Image-2
5. **画风扩展**：本文为3D国漫仙侠风。如需AI真人风/2D国风/横版等其他画风，需独立维护提示词库
6. **GPT-Image注意事项**：文中使用了GPT-Image-2生成。但输出质量受模型版本影响，提示词可能需要微调
