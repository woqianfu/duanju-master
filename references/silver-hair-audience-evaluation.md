# 银发受众评估方法（银发大师）

> 基于《代码里的长安》60集v4.0实战验证
> 中老年受众(50岁+)占短剧大盘27.5%

## 适用场景

对剧本/成片进行「银发友好度」评估时调用此方法。银发群体（50岁+）偏好：家庭温情、历史古风、年代记忆、文化传承类内容。

## 五维评估体系

### 维度1：叙事节奏

| 标准 | 说明 |
|------|------|
| 每集时长 | 银发受众平均观看时长90-120秒（高于大盘60秒），建议银发向集延长至90秒 |
| 情境建立 | 前15秒可"慢热"铺垫，但必须有清晰的人物关系和故事主线 |
| 场景切换 | 避免过快切换（3秒以内），50+观众需要时间建立空间感 |
| ✅ 通过标准 | 前15秒内让观众知道「谁在哪要干嘛」 |

### 维度2：文化认知契合度

| 高共鸣元素 | 说明 |
|-----------|------|
| 历史题材 | 敦煌、唐代、书法、传统乐器——对50+观众有天然吸引力 |
| 传统美学 | 水墨画、笛子、古风服饰——情感锚点 |
| 代际传承 | 外婆/奶奶线、传家宝、遗物、未完成的心愿——最高共鸣主题 |
| 跨代情感 | 孙辈理解长辈的遗志——银发观众最容易代入的情感通道 |

### 维度3：对话与旁白可理解度

| 规则 | 说明 |
|------|------|
| 旁白接受度 | 银发观众对叙事性旁白的接受度高于年轻观众（可占集时长30%） |
| 旁白语速 | 建议控制在每分钟200字以内 |
| 术语处理 | 专业术语（如多光谱扫描、递归坐标等）必须通过角色对话自然解释，不可用旁白独白交代 |
| 画面>台词 | 重要的信息/情感传递优先用画面+动作，而非对话说明 |

### 维度4：情感共鸣度

| 高共鸣情感点 | 实战验证 |
|-------------|---------|
| 守望主题 | 等一个人一千二百年——对银发观众有极强感染力 |
| 代际保护 | 老一辈发现秘密→被灭口→留下线索给下一代——高共鸣结构 |
| 专业遗憾 | "一辈子没做出自己的颜色"——老年人在专业上的遗憾是泪点 |
| 情感释放时长 | 关键情感高潮场景建议延长至90秒，给银发观众充分的情感释放时间 |

### 维度5：视觉清晰度

| 标准 | 说明 |
|------|------|
| 字幕字号 | 不小于36pt（手机端竖屏） |
| 关键信息呈现 | 坐标、书信文字、配方内容等——用加粗大字号+声音朗读双重呈现 |
| 明暗对比 | 银发观众对低亮度画面辨识度下降，特效场景（如密室发光、水墨效果）需加强明暗对比 |
| ✅ 通过标准 | 关掉声音，光看画面能看懂剧情走向 |

## 银发友好度检查清单

完成五维评估后，逐项检查：

- [ ] 前15秒让观众知道「谁在哪要干嘛」
- [ ] 50+观众能理解全部专业术语（通过角色对话自然解释）
- [ ] 字幕36pt+，关键信息双重呈现
- [ ] 情感高潮场景预留90秒释放时间
- [ ] 至少有1个跨代情感连接点（祖孙/代际传承）
- [ ] 去掉台词，光看画面能看懂

## 银发大师综合评分

| 维度 | 权重 | 评分标准 |
|------|:----:|:---------|
| 叙事节奏 | 25% | A=节奏舒适的90秒集/ B=标准的60秒集/ C=过快要调整 |
| 文化认知 | 20% | A=历史古风高共鸣/ B=题材中性/ C=题材不对标 |
| 对话理解 | 20% | A=无术语障碍/ B=术语有解释/ C=术语未处理 |
| 情感共鸣 | 25% | A=有泪点且释放充分/ B=有泪点但仓促/ C=无代际情感点 |
| 视觉清晰 | 10% | A=字体对比度均达标/ B=部分达标/ C=不达标 |
| **综合** | 100% | A=银发友好/B=可优化/C=需重做 |

## 实战参考

《代码里的长安》银发大师评分为 **B+**：
- 叙事节奏 C+（60秒分割偏快）→ 建议部分集延长至90秒
- 文化认知 A（敦煌+唐代+书法+笛子=银发高共鸣）
- 对话理解 B（部分术语需角色自然解释）
- 情感共鸣 A-（外婆线+等待主题→高泪点）
- 视觉清晰 B（需加大字幕+提高明暗对比）

修复方向：
1. E45（外婆画出延炎）和E60（笛子刻字+外婆铅笔痕）两个情感高点延长至90秒
2. 专业术语（递归坐标、多光谱、叠层校验）通过角色对话自然分步解释
3. 字幕在拍摄时标注36pt（当前剧本仅标注内容，未标注字体规格）
